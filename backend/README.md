# Backend API

This is a NestJS backend application for the PW Genius registration system.

## Features

- Registration API with file upload support
- S3 file storage integration
- MongoDB integration for storing registration logs
- RESTful API endpoints

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure MongoDB is running:
```bash
brew services start mongodb-community
```

3. Start the development server:
```bash
npm run start:dev
```

## API Endpoints

### Registration
- `POST /register` - Submit a new registration with optional file upload
- `GET /register/logs` - Get all registration logs
- `GET /register/logs/:id` - Get a specific registration log
- `DELETE /register/logs/:id` - Delete a registration log

## Database

The application uses MongoDB to store registration logs. Each registration submission is automatically saved to the database with the following information:

- All form data (student name, class, parent names, etc.)
- File upload details (if applicable)
- S3 file URL (if file was uploaded)
- Timestamps
- Status (success/error)
- Error messages (if any)

## Environment Variables

Make sure to configure your S3 credentials and other environment variables as needed.
